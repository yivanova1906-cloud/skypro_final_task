Git

  • git init — сокращение от “initialize”; команда инициализирует новый Git-репозиторий в текущей папке.
  • git status — команда “status”; показывает текущее состояние репозитория (какие файлы изменены, что готово к коммиту и т. п.).
  • git log — команда “log”; выводит историю коммитов (кто, когда и что закоммитил).
  • git branch — команда “branch”; показывает список веток и/или создаёт новую ветку.
  • git clone — команда “clone”; скачивает удалённый репозиторий на компьютер и автоматически настраивает remote origin.​
